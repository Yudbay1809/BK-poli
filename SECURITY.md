# Security Policy

## Supported Versions

Hanya branch `main` yang saat ini didukung untuk patch keamanan.

## Reporting a Vulnerability

Jangan laporkan celah keamanan lewat issue publik.

Gunakan salah satu kanal berikut:
1. GitHub Security Advisory (preferred)
2. Email: `prastyodx2@gmail.com`

Sertakan informasi berikut agar cepat ditangani:
1. Ringkasan masalah dan dampaknya.
2. Langkah reproduksi.
3. Bukti konsep (PoC) jika ada.
4. Versi/commit yang terdampak.

Target respons awal: maksimal 72 jam.
